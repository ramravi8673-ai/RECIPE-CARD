# 🍽️ Recipe Card

A responsive and interactive Recipe Card web application built using **HTML, CSS, and JavaScript**. This project showcases a clean, modern recipe card interface with an emphasis on responsive design, usability, and front-end development best practices.

## 🚀 Features

* 📱 Fully responsive design
* 🎨 Clean and modern user interface
* ⚡ Interactive functionality using JavaScript
* 📝 Well-structured recipe information
* 🌐 Fast-loading static website
* 💻 Beginner-friendly and lightweight

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
recipe-card/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🎯 Learning Objectives

This project was created to strengthen core front-end development skills by practicing:

* Semantic HTML
* Responsive CSS layouts
* DOM manipulation with JavaScript
* Clean UI design
* Project deployment using Vercel
